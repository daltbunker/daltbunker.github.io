Soul Climbers

Soul Climbers is a tool and hub for "pure" rock climbers. Users can explore news articles, search for climbs, and log their ascents without the hastle of gym bros and jaded trad dads. Other popular sites tend to only distract from the beauty and truths or rock climbing. As a recent converted soul climber, this is my way of giving back. (Java, MySQL, Angular)

Media Storage

This app can host a collection of images and videos for a user. Users can effortlessly sort and search through their libaray of media. Authentication is handled with OAuth 2.0 and JWT web tokens. (Go, React, Postgres)

9a Database

Open source project to host all rock climbs graded 9a or harder. Built with GraphQL for simple and customizable data access. React interface to explore data and sample api usage. (Postgres, GraphQL, React)

Task Assistant

A tool to help manage your coding projects and stay organized. Create tasks with different priorities and see where you're spending the most time. Explore data visulations built with D3.js. (React, Node, MongoDB)
