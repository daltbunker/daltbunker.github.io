Soul Climbers

Soul Climbers is a tool and hub for "pure" rock climbers. Users can explore news articles, search for climbs, and log their ascents without the hastle of gym bros and jaded trad dads. Other popular sites tend to only distract from the beauty and truths or rock climbing. As a recent converted soul climber, this is my way of giving back. (Java, MySQL, Angular)

Chat Server

This api can be used to spin up concurrent sockets to allow for asyncrhonous communication between users. I built this project as a sort of playground to explore the WebSocket protocol and the popular concurrency features of go. Namely go routines and channels. (Go)

Task Assistant
